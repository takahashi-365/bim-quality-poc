## 8. 応募先タイプ別に強調するポイント

### BIMマネージャー / BIM推進 / BIM導入支援寄り

強調すること：

- BIM導入支援経験
- Revitコンサルティング経験
- BIM運用支援
- BIMデータ品質管理
- パラメータ未入力、分類コード未入力、命名規則違反の整理
- Revit API / pyRevit連携は今後の拡張候補

あまり強く出しすぎないこと：

- AIモデル
- 機械学習精度
- 生成AI

---

### 建築DX / データ活用 / BI寄り

強調すること：

- BIMデータを分析可能な構造化データへ整える力
- Python/pandasによるデータ処理
- CSV処理
- 品質メトリクス作成
- Power BI / Streamlit可視化
- データクレンジング
- 品質集計・スコア化

あまり強く出しすぎないこと：

- Revit API実装済みのような表現
- 生成AIで自動判断できるような表現

---

### AI活用 / 生成AI / PoC開発寄り

強調すること：

- PythonによるPoC開発
- 特徴量データセット作成
- scikit-learnによる分類プロトタイプ
- 生成AI向け構造化コンテキスト生成
- AIに渡す情報をRuleIdベースで制御する設計
- 実務適用には教師データや追加検証が必要であるという理解

あまり強く出しすぎないこと：

- 高精度AIモデルを作った
- 生成AIでBIMモデルを自動修正できる
- 実務用AIシステムを完成させた
