# Local LLM Explanation Demo

This folder contains sample inputs and outputs for the Local LLM Explanation Demo.

This experiment uses AI Context v002 and Fix Guide Markdown generated by the BIM Data Quality & AI Readiness Assessment PoC.

The goal is not to let the LLM make design or construction decisions.
The goal is to check whether structured BIM quality context can be transformed into human-readable explanations for BIM staff.

Model files, local cache, logs, confidential data, and API keys are not included.

## Japanese Note

このフォルダは、Local LLM Explanation Demo の入力サンプルと出力例を整理するためのものです。

本実験では、第2段階PoCで生成した AI Context v002 と Fix Guide Markdown を利用します。

目的は、LLMに設計判断や施工判断をさせることではありません。
構造化されたBIM品質コンテキストを、BIM担当者が確認しやすい説明文へ変換できるかを確認することです。

LLMの回答は参考情報であり、最終判断はBIM担当者が行う前提です。

モデル本体、ローカルキャッシュ、ログ、機密データ、APIキーは含めません。
